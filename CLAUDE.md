# Project Guidelines

## Git Workflow
- After completing each meaningful task, automatically commit changes
- Use descriptive commit messages in format: 	ype: description
  - feat: new feature
  - fix: bug fix
  - docs: documentation
  - refactor: code refactoring
  - chore: maintenance
- Push to origin/main after each commit
- Always pull before push to avoid conflicts
