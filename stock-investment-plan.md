### 3.5 市场情绪与新闻分析

> 情绪是市场的温度计。正确的情绪判断能显著提升仓位管理的有效性，避免在极端情绪下做出错误决策。

#### 3.5.1 情绪指标体系

| 指标类别 | 具体指标 | 数据来源 | 解读方法 |
|---------|---------|---------|---------|
| **涨跌停对比** | 涨停家数 vs. 跌停家数 | 东方财富/同花顺 | 涨停远多于跌停 = 多头主导；反之 = 恐慌蔓延 |
| **量比** | 当日成交量 / 5日平均成交量 | 行情软件 | >1.5 = 放量活跃；<0.8 = 缩量冷清 |
| **融资余额变化** | 两融余额日环比变动 | 交易所每日披露 | 持续增加 = 杠杆做多意愿增强；骤降 = 去杠杆恐慌 |
| **北向资金流向** | 沪股通+深股通净买入/卖出 | 东方财富/同花顺 | 连续大幅净流入 = 外资看多；大幅流出 = 风险偏好下降 |
| **期指升贴水** | 当月合约 vs. 现货指数价差 | 期货行情 | 升水（期货>现货）= 多头乐观；贴水 = 空头预期增强 |
| **市场宽度** | 上涨家数 / 下跌家数 | 行情软件 | >2 = 普涨强势；<0.5 = 普跌弱势 |

#### 3.5.2 消息面分类框架

| 类别 | 子类 | 典型来源 | 影响周期 |
|-----|------|---------|---------|
| **政策类** | 货币政策（降准/降息/MLF/LPR） | 央行公告 | 中长期 |
| | 资本市场监管（IPO节奏、减持新规） | 证监会公告 | 中期 |
| | 产业/财政政策（补贴、税收优惠） | 国务院/发改委 | 中期 |
| **业绩类** | 财报超预期 | 上市公司公告 | 短期（1-3天） |
| | 财报低于预期/暴雷 | 上市公司公告 | 短期（负面持续） |
| **行业政策** | 行业补贴与扶持 | 行业主管部门 | 中期 |
| | 行业监管与整顿 | 行业主管部门 | 短-中期（负面） |
| **宏观经济** | CPI/PPI、PMI、社融、M2 | 国家统计局/央行 | 中长期 |
| | 外贸数据、进出口 | 海关总署 | 短-中期 |
| **地缘政治** | 关税政策、贸易摩擦 | 商务部/外交部 | 短-中期 |
| | 人民币汇率波动 | 外汇交易中心 | 短期 |

#### 3.5.3 情绪量化打分（1-5 分）

| 得分 | 情绪状态 | 典型特征 |
|:----:|---------|---------|
| **1** | 极度恐慌/冰点 | 涨停<5家，跌停>50家；量比>2（恐慌放量）；融资余额骤降；北向大幅流出；期指深度贴水；市场宽度<0.3 |
| **2** | 偏弱 | 跌停多于涨停；量比0.8-1.2；融资余额小幅下降；北向小幅流出；期指小幅贴水；市场宽度0.5-0.8 |
| **3** | 中性 | 涨跌停家数相近；量比0.8-1.2；融资余额平稳；北向小幅波动；期指平水附近；市场宽度0.8-1.2 |
| **4** | 偏强 | 涨停远多于跌停；量比1.2-1.5；融资余额增加；北向持续流入；期指升水；市场宽度1.5-2.0 |
| **5** | 极度亢奋/泡沫 | 涨停>100家，跌停<5家；量比>2（亢奋放量）；融资余额激增；北向大幅流入；期指深度升水；市场宽度>2.5 |

#### 3.5.4 情绪与仓位关系

| 情绪得分 | 仓位上限 | 说明 |
|:--------:|:--------:|------|
| 1 分（冰点） | **≤15%** | 市场极度恐慌，流动性风险高，仅可在恐慌衰竭信号出现时小仓试探 |
| 2 分（偏弱） | **≤15%** | 市场弱势，反弹为减仓机会，严守止损 |
| 3 分（中性） | **≤25%** | 正常交易环境，可按评分模型正常选股交易 |
| 4 分（偏强） | **≤20%** | 市场偏热，注意追高风险，优先获利了结 |
| 5 分（亢奋） | **≤20%** | 泡沫信号，过热反而减仓，不追高，准备反向做空或空仓回避 |

> ⚠️ **核心原则：情绪极端时（1分或5分），市场反转概率显著增加。冰点酝酿机会，但需等待确认；亢奋孕育风险，应主动收缩仓位。**

## 4. 选股标准

### 4.1 股票池筛选

**主要市场：A 股（沪深）**

| 筛选条件 | 标准 | 理由 |
|---------|------|------|
| 市值 | ¥50亿 – ¥1000亿 | 流动性充足，不过于迟缓 |
| 日均成交额 | > ¥1亿 | 确保进出流动性 |

| ST 状态 | 无 ST/*ST | 避免退市风险 |
| 上市时间 | > 6 个月 | 避免 IPO 波动 |
| 融资融券 | 符合条件 | 增加灵活性 |

### 4.2 评分模型（总分 100 分）

| 因素 | 权重 | 评分标准 |
|-----|-----:|---------|
| **趋势一致性** | 25 分 | MA5 > MA20 > MA60 = 25；部分一致 = 15；不一致 = 5 |
| **成交量确认** | 20 分 | 上涨日成交量 > 1.5 倍均值 = 20；中等 = 12；弱势 = 5 |
| **板块动能** | 20 分 | 前 3 板块 = 20；前 5 = 14；中性 = 8 |
| **基本面催化剂** | 15 分 | 业绩超预期/政策利好 = 15；中性 = 8；负面 = 3 |
| **技术形态** | 10 分 | 突破/支撑位反弹 = 10；中性 = 5；破位 = 2 |
| **风险回报比** | 10 分 | R:R > 3:1 = 10；> 2:1 = 7；> 1:1 = 3 |

**入场门槛：得分 ≥ 70/100**

## 5. 风险管理规则

### 5.1 硬性规则（绝不违反）

| 规则 | 限制 | 后果 |
|-----|------|------|
| 单一持仓上限 | 资金的 30% | 超过自动减仓 |
| 日内最大亏损 | **¥200**（目标的 2.5 倍） | **当日停止交易** |
| 周最大亏损 | **¥500** | 下周仓位减半 |
| 峰值最大回撤 | 10% | 停止所有交易，全面复盘 |
| 单笔止损 | 入场价的 2% | 不可商量，市价单离场 |
| 隔夜融资 | 0% 融资过夜 | 14:50 前平掉所有杠杆仓位 |

### 5.2 软性规则（需判断）

- 避免开盘前 15 分钟交易（9:30–9:45）—— 高波动噪音
- 避免收盘前 10 分钟交易（14:50–15:00），除非平仓
- 政策重大公告日若方向不明则不交易
- **重大负面新闻/政策出台日不交易**：若出现重大负面新闻或突发政策（如加征关税、行业整顿、监管收紧），且市场方向不明，当日停止所有交易，避免政策真空期的极端波动
- 连续 2 笔亏损后将仓位减少 50%
- 连续 3 笔亏损后休息 30 分钟

### 5.3 亏损恢复计划

| 日内亏损 | 次日行动 |
|---------:|:--------|
| < ¥100 | 正常交易 |
| ¥100–200 | 仓位减至 50%，优先回本 |
| > ¥200 | 不交易，复盘与重置 |
| 连续 3 天亏损 | 全天休息，日志复盘 |

## 6. 仓位管理

### 6.1 固定比例模型

```
仓位规模 = (单笔账户风险) / (入场价 × 止损比例)

其中：
  单笔账户风险 = 总资金的 1% = ¥200（以 ¥20,000 计）
  止损 = 入场价的 2%

示例：
  股价 = ¥40
  止损 = 2% = ¥0.80
  账户风险 = ¥200
  仓位规模 = ¥200 / ¥0.80 = 250 股
  仓位价值 = 250 × ¥40 = ¥10,000（资金的 50%）
```

### 6.2 仓位分级系统

| 交易质量 | 得分 | 仓位规模 | 资金配置 |
|---------|:----:|:--------|:--------|
| A+ 形态 | 85+ | 25–30% | ¥5,000–6,000 |
| A 形态 | 75–84 | 20–25% | ¥4,000–5,000 |
| B+ 形态 | 70–74 | 15–20% | ¥3,000–4,000 |
| 低于门槛 | <70 | **不交易** | ¥0 |

### 6.3 分批进出

- **分批建仓**：信号出现时入场 50%，回调确认加仓 25%，突破加最后 25%
- **分批退出**：1R 利润卖出 50%，止损移至盈亏平衡，2R 卖出 25%，移动止损持有最后 25%

## 7. 离场策略

### 7.1 止盈目标

| 目标级别 | 价格目标 | 操作 |
|---------|---------|------|
| 1R（初始风险） | 入场价 + 止损距离 | 卖出 50%，止损移至盈亏平衡 |
| 2R | 入场价 + 2 × 止损距离 | 卖出 25%，移动止损 |
| 3R+ | 以 1% 移动止损跟踪 | 让利润奔跑 |
| 时间止损 | 3 天内波动 < 1% | 全部清仓 |

### 7.2 止损规则

| 止损类型 | 触发条件 | 操作 |
|---------|---------|------|
| 初始止损 | 入场价下方 2% | 全部离场，市价单 |
| 移动止损 | 入场后最高价下方 1% | 卖出剩余仓位 |
| 时间止损 | 3 天，价格 < 入场价 + 1% | 全部离场 |
| 跳空止损 | 开盘低于前收盘价 3% 以上 | 开盘市价卖出 |
| 新闻止损 | 重大负面催化剂 | 立即离场 |

### 7.3 日内盈利目标

| 里程碑 | 操作 |
|-------|------|
| 达到 ¥80 利润 | 当日平掉所有仓位 ✅ |
| ¥120 利润 | 考虑收盘 —— 状态极佳 |
| ¥150+ 利润 | 必须收盘 —— 不要回吐 |

> **核心原则**：达到日内目标后，停止交易。贪婪是稳定性的敌人。

## 10. 策略手册 —— 3 个核心形态

### 形态 1：早盘突破（9:45–10:15）

- **条件**：股价高开 1-3%，前 15 分钟成交量 > 2 倍均值
- **入场**：初始拉升后回踩 VWAP 或 MA5 时买入
- **止损**：开盘价下方或 -2%，取更紧者
- **目标**：1R–2R
- **得分要求**：75+

### 形态 2：板块轮动交易（10:00–11:00）

- **条件**：板块突然涌入资金，板块内龙头股突破
- **入场**：首次回踩 MA5 时买入板块龙头
- **止损**：MA20 下方或 -2%
- **目标**：2R–3R（板块行情有持续性）
- **得分要求**：80+

### 形态 3：午后反转（13:30–14:30）

- **条件**：上升趋势中的股票回踩 MA20 支撑，RSI < 35，回调时成交量萎缩
- **入场**：价格在 MA20 处反弹且成交量放大时买入
- **止损**：MA20 下方 1% 或 -2%
- **目标**：1R–2R（快进快出，不要恋战）
- **得分要求**：70+

## 11. 心理与纪律规则

### 11.1 五大戒律

1. **不可追高** —— 错过入场点，就等下一个
2. **不可补仓摊低成本** —— 止损就是止损，不是加仓理由
3. **不可报复性交易** —— 亏损后要休息，不是再交易
4. **不可违背日内目标** —— 达到 ¥80？结束交易。离场。
5. **不可遗漏交易日志** —— 无例外，不要"我会记住"

### 11.2 情绪状态检查

每笔交易前，评估你的情绪状态：

| 状态 | 评分 | 行动 |
|-----|:----:|------|
| 平静、专注 | 1–2 | ✅ 正常交易 |
| 略微焦虑/急切 | 3 | ⚠️ 仓位减少 25% |
| FOMO / 挫败 / 愤怒 | 4–5 | 🛑 不要交易 |

### 11.3 休息计划

- 连续 2 笔亏损后：休息 15 分钟
- 触及日内亏损上限后：当日停止交易
- 触及周度亏损上限后：全面复盘，下周减仓
- 情绪失控（评分 4+）：至少离开 30 分钟
